Web Hosting

You have your site built. Now what? To get it on the web, you need a host. Some people prefer to set up and manage their own hosts — if that’s you, I’ll supply you with your website code in a zip file that you can upload and extract to your host's servers.

If you’d rather not deal with the hassle of setting up the host yourself, I'll do it for you. Once I've set it up, I’ll also maintain the hosting account and resolve any outages or other technical issues that occur.
